# 詐欺電話撃退アプリ

🛡️ 電話中の詐欺のサインをその場でチェック、通話後は録音をAIが詐欺判定するPWAです。

▶️ 今すぐ試す：https://tomonori2.github.io/sagi-keikoku/

（インストール不要。ブラウザで開くだけ。通話後の録音判定だけ、無料のGroq APIキーを自分で用意して使います）

> 詐欺の被害額は、数万円で済むこともあれば、数千万円に達することもあります。
> このアプリは無料です。通話中に開くだけで詐欺のサインをその場でチェックでき、録音機能がないスマホでも注意喚起として役立ちます。
> 入れておくだけで意味がある、そんなお守りアプリです。

## できること

- 📋 通話中に見せる「詐欺のサインチェックリスト」（オレオレ詐欺・還付金詐欺・ニセ警察官詐欺・ロマンス詐欺など）
- 📞 迷ったらワンタップで警察相談専用電話（#9110）
- 🎙️ 通話後、録音ファイルをAIが文字起こし＆危険度判定（高／中／低）
- 📵 危険な電話番号の記録・検索（次に同じ番号からかかってきたら警告）
- 📰 警察庁の最新注意喚起を毎朝自動取得し、AIの判定材料に反映

## 使い方

1. **通話中**：怪しいと思ったらアプリを開くだけ。設定不要。
2. **通話後**：スマホの録音ファイルを開いて「共有」→「詐欺撃退」を選ぶだけ。自動で文字起こし＆AI判定が始まります（Androidの共有機能を利用）。

②だけ、[console.groq.com/keys](https://console.groq.com/keys) で無料のAPIキーを発行して、アプリ内の「設定」から登録してください。

**②を使うには先にアプリをホーム画面に追加（インストール）しておく必要があります**（ブラウザのメニュー→「ホーム画面に追加」）。インストールしていないと共有メニューに出てきません。対応はAndroid＋Chrome系ブラウザです（iPhoneのSafariはWeb Share Target非対応のため②は使えません）。

## 技術構成

- サーバー無し・ビルド無し。`index.html` 1枚 + `sw.js` + `manifest.json` のみ
- 文字起こし・詐欺判定は Groq API（Whisper・Llama 3.3）をブラウザから直接呼び出し
- APIキーはブラウザの `localStorage` にのみ保存（外部送信なし）
- Web Share Target 機能で、電話アプリなどから録音ファイルを「共有」で直接受け取れる
- `news_data.js`：警察庁SOS47ページから `fetch_scam_news.py` が毎朝自動取得（Windowsタスクスケジューラ）
- `curated_news.js`：運営が確認して追加した情報（会話ベースで更新・全ユーザー共通）
